Приложение для Женской консультации.

Описание:
Это приложение, в котором собраны все поля, из вордовских шаблонов с возможностью выгрузить и загрузить из ворда.

Меню:
1. Новый протокол
	Очищает все поля в программе. Нажмите на эту кнопку, когда приходит новый пациент
2. Открыть протокол
	По пути, указанному в настройках, хранятся все протоколы. Из списка протоколов можно выбрать нужный.
	Также работает поиск по названию. Буквы могут быть строчными или заглавными, для поиска это не имеет значения.
3. Сохранить
	По пути, указанному в настройках, создаётся папка. Название папки - ФИО пациента. Внутрь складываются файлы протоколов по этому пациенту.
4. Сохранить как
	Можно создать папку для сохранения самим и выбрать её.
5. Выход
6. Настройки
	- Базовые настройки
		Тут можно добавить нового врача, а таже выбрать базовую директорию.
		ВАЖНО: Для того, чтобы новый врач отобразился в списке выбора нужно лиюо создать новый протокол, либо открыть нужный.
		В базовой директории создаются папки протоколов и поиск ранее созданных протоколов производится именно в этой папке
	- Перцентильные коридоры
		Тут можно посмотреть данные перцентильных коридоров для фетометрии, а также изменить их значения.
		ВАЖНО: Для того, чтобы новое значение сохранилось, надо перевести курсор на другое поле
	- Значения констант
		Тут написана формула, по которой рассчитывается масса плода. Также здесь можно поменять значения констант формулы

Основное окно:
1. Общие данные
	Тут можно заполнить ФИО, возраст, дату последней менструации пациента. Также здесь находятся кнопки для увеличения и уменьшения числа плодов.
	Для этого нужно сделать активной вкладку с нужным плодом и нажать кнопку "Удалить плод". 
2. Данные о триместрах
	Рядом со сроком беременности есть кнопка в форме календаря с ручкой, которая рассчитает срок беременности по дате 
	последней менструации.

	ВАЖНО: Чтобы данные текстового поля обновились, нужно перевести курсор на другое поле
3. Фетометрия
	В первом триместре там есть кнопка "Рассчитать перцентильные коридоры". По её нажатии заполняется поле нед. рядом с каждым параметром
	Эта неделя - срок, в котором такие размеры были бы нормой
	Например: сейчас идёт 15я неделя беременности, а бипариетальный размер малыша 35мм. Норма для этого параметра на 15й неделе 31-33.
			Зато на 16й неделе 35мм было бы нормой. Поэтому в поле нед. встанет число 16.

	В втором и третьем триместре есть кнопка "Рассчитать массу и проверить перцентильные коридоры".
	Функционал этой кнопки является расширением функционала кнопки "Рассчитать перцентильные коридоры" из первого триместра, поэтому
	поле нед. заполняется по такому же принципу. Также по этой кнопке рассчитывается масса плода по формуле, которую можно
	найти в настройках. И ещё заполняется больше или меньше нормы данный показатель.


Какие фичи добавляли:
1. При создании нового протокола автоматом ставится текущая дата
2. При сохранении, если папка с таким имененм есть, программа просит ввести новое название и предлагает вариант по дефолту
